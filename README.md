# pv-module-layout-designer-app
A MATLAB App Designer GUI tool to design and visualize custom photovoltaic (PV) module layouts.

Included features:
1) Real time update of 3D plot according to the values entered
2) Flexibility to choose the type of view
3) Option to view each component individually
4) Ability to see layers stacked upon each other with clear visibility
5) User can download the present configuration as a json file
6) User can open and work with an existing module configuration json file 
7) Input validity checks to ensure that entered values are legit
8) Option to reset all the values to default values
